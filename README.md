# reefbeat⚡Backup

Système autonome de monitoring et de gestion de batterie de secours pour aquarium récifal Red Sea (ReefWave, ReefRun, ReefLED, Skimmer DC).

Autonomous backup battery monitoring and management system for Red Sea reef aquariums.

## ⚡ Features

- **Battery monitoring** via INA226 (I2C) or Victron Blue Smart IP22 (BLE)
- **Instant outage detection** via 230V relay on GPIO
- **Progressive pump intensity** — configurable SoC-based levels (eco → survival → critical)
- **Per-device control** — set individual speeds for each ReefWave/ReefRun/ReefLED
- **3-level network failover** — direct reach → Wi-Fi rejoin → mirror hotspot
- **Home Assistant integration** — MQTT auto-discovery with 10 sensors
- **Auto-detection** — scans your network for ReefBeat devices during setup
- **Bilingual** — French/English interface based on system locale

## 🚀 Quick Install

```bash
curl -sL https://raw.githubusercontent.com/Elwinmage/reefbeatEnergyBackup/main/install.sh | sudo bash
```

The installer will:
1. Download the latest release
2. Install system and Python dependencies
3. Launch an interactive configuration wizard that:
   - Scans your network for ReefBeat devices
   - Retrieves Wi-Fi SSID and MAC addresses from your devices
   - Configures battery, monitoring, pump levels, and MQTT

## 📁 Project structure

```
install.sh              One-line installer (curl | bash)

```

## 🔧 Hardware

### Battery monitoring (choose one)

**INA226 (I2C module)**

| INA226 | RPi 3 |
|--------|-------|
| VCC | Pin 1 (3.3V) |
| GND | Pin 6 (GND) + battery negative |
| SDA | Pin 3 (GPIO2) |
| SCL | Pin 5 (GPIO3) |
| IN+ | Battery + (after fuse) |
| IN- | Load + (to pumps) |

**Victron Blue Smart IP22 (BLE)**

No wiring needed — reads data wirelessly via Bluetooth Low Energy.
Encryption key required from VictronConnect app (Settings → Product Info → Instant Readout).

### Outage detection

**Finder 40.61.8.230.4000 relay**

| Relay | Connection |
|-------|------------|
| Coil A1/A2 | 230V Phase + Neutral |
| NO contact | GPIO pin → GND (with 10kΩ pull-up to 3.3V) |

### Power architecture

```
Mains 230V → LiFePO4 24V Charger → Battery 24V → ReefWave / ReefRun / Skimmer
                                         ↓
                                    RPi 3 (via 24V→5V DC-DC)
                                         ↓
                                    INA226 (monitoring)
```

## 🏠 Home Assistant sensors

| Entity | Description |
|--------|-------------|
| `sensor.reef_battery_voltage` | Battery voltage (V) |
| `sensor.reef_battery_current` | Current (A, + = discharging) |
| `sensor.reef_battery_power` | Power (W) |
| `sensor.reef_battery_soc` | State of Charge (%) |
| `sensor.reef_battery_power_state` | mains / battery |
| `sensor.reef_battery_pump_intensity` | Average pump intensity (%) |
| `sensor.reef_battery_pump_level` | Active level name |
| `sensor.reef_battery_runtime` | Estimated runtime (h) |
| `sensor.reef_battery_outage_duration` | Outage duration (min) |
| `sensor.reef_battery_network_mode` | client / rejoin / hotspot |
| `sensor.reef_battery_monitor_source` | ina226 / victron |
